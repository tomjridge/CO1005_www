# Java tutorials You need to know the basics of Java before taking this course. Some resources are: * [Java tutorials at Oracle (see especially the 'getting started' trail)](http://docs.oracle.com/javase/tutorial/index.html) * [Introduction to Programming Using Java](http://math.hws.edu/javanotes/index.html) A nice introduction * [Introduction to Programming in Java](https://ocw.mit.edu/courses/electrical-engineering-and-computer-science/6-092-introduction-to-programming-in-java-january-iap-2010/lecture-notes/) Basic course from MIT * [Intro by Bradley Kjell](http://chortle.ccsu.edu/cs151/cs151java.html) Lots of exercises * [Cafe au lait](http://www.cafeaulait.org/course/) Comprehensive on the basics (e.g. jar files) * [Princeton Java intro](http://introcs.cs.princeton.edu/java/home/) Good
